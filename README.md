# This application makes a request to the API of the resource "https://openweathermap.org/" and receives weather data.

### Homework #2

## Setup

```bash
gradle clean build
```
OR
```bash
make build
```

## Run Application

```bash
gradle run
```
OR
```bash
make start
```

## Run tests

```bash
gradle test
```
OR
```bash
make test
```

## Run linter

```bash
gradle checkstyleMain checkstyleTest
```
OR
```bash
make lint
```
